üìã STEP 1: GIT INSTALL KARO
----------------------------
1. Git download karo: https://git-scm.com/download/win
2. Install karo (default settings rakho)
3. Restart karo computer ko (optional but recommended)


üìã STEP 2: GITHUB ACCOUNT BANAO
--------------------------------
1. https://github.com pe jao
2. Sign up karo (free account)
3. Email verify karo


üìã STEP 3: GIT CONFIGURE KARO (CMD mein ye commands run karo)
--------------------------------------------------------------
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

(Apna naam aur email dalo jo GitHub pe use kiya hai)


üìã STEP 4: PROJECT FOLDER MEIN GIT INITIALIZE KARO
---------------------------------------------------
Apne project folder mein CMD open karo aur ye commands run karo:

cd C:\path\to\your\project
git init
git add .
git commit -m "Initial commit - Minecraft Plugin Store"


üìã STEP 5: GITHUB PE REPOSITORY BANAO
--------------------------------------
1. GitHub pe login karo
2. Top right corner mein "+" click karo
3. "New repository" select karo
4. Repository details:
   - Name: minecraft-plugin-store (ya koi bhi naam)
   - Description: Minecraft Plugin Store with React & Flask
   - Public ya Private (tumhari choice)
   - ‚ùå README, .gitignore, license ADD MAT KARO (already hai)
5. "Create repository" click karo


üìã STEP 6: LOCAL PROJECT KO GITHUB SE CONNECT KARO
---------------------------------------------------
GitHub pe naya repo banane ke baad ye commands dikhenge, copy karo aur CMD mein paste karo:

git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
git branch -M main
git push -u origin main

(YOUR-USERNAME aur YOUR-REPO-NAME apne se replace karo)


üìã STEP 7: GITHUB USERNAME/PASSWORD ENTER KARO
-----------------------------------------------
Jab push karoge to GitHub credentials mangega:
- Username: Apna GitHub username
- Password: ‚ùå Password NAHI, Personal Access Token chahiye!

Personal Access Token banao:
1. GitHub pe jao ‚Üí Settings ‚Üí Developer settings
2. Personal access tokens ‚Üí Tokens (classic)
3. "Generate new token" ‚Üí "Generate new token (classic)"
4. Note: "Git Access"
5. Expiration: 90 days (ya No expiration)
6. Select scopes: ‚úÖ repo (pura check karo)
7. "Generate token" click karo
8. Token COPY karo (ye sirf ek baar dikhega!)
9. Is token ko password ki jagah use karo


üìã STEP 8: VERIFY KARO
-----------------------
1. GitHub pe apni repository kholo
2. Files dikhengi (videos ke alawa sab kuch)
3. ‚úÖ Success!


üé• VIDEO FILES KA SOLUTION
===========================

Option 1: CLOUDINARY (Recommended - Free & Easy)
-------------------------------------------------
1. https://cloudinary.com pe account banao
2. Videos upload karo

--------------------------------------
1. GitHub repo mein "Releases" section pe jao
2. "Create a new release" click karo
3. Videos ko assets mein upload karo
4. Release publish karo
5. Video URLs code mein update karo


Option 3: AWS S3 / Google Drive / Dropbox
------------------------------------------
Videos ko cloud storage pe upload karo aur public URLs use karo

‚ùå COMMON ERRORS & SOLUTIONS
=============================

Error 1: "fatal: not a git repository"
Solution: git init run karo


Error 2: "failed to push some refs"
Solution: 
git pull origin main --rebase
git push origin main


Error 3: "Permission denied"
Solution: Personal Access Token use karo password ki jagah


Error 4: "large files"
Solution: .gitignore check karo, videos ignore ho rahe hain ya nahi


üéØ QUICK COMMANDS REFERENCE
============================

Check status:           git status
See changes:            git diff
View commit history:    git log
Undo last commit:       git reset --soft HEAD~1
Remove file from git:   git rm --cached filename
Update .gitignore:      git rm -r --cached . && git add . && git commit -m "Update gitignore"
