STEP 1: Git Initialize Karo
----------------------------
Terminal mein yeh commands run karo:

cd C:\Users\YourUsername\your-project-folder
git init
git add .
git commit -m "Initial commit - Minecraft Plugin Store"


STEP 2: GitHub Pe New Repository Banao
---------------------------------------
1. Browser mein jao: https://github.com
2. Login karo
3. Top right corner mein "+" click karo
4. "New repository" select karo
5. Repository name do: minecraft-plugin-store
6. Description (optional): "A Minecraft plugin marketplace"
7. Public ya Private choose karo
8. ‚ùå README, .gitignore, license mat add karo (already hai)
9. "Create repository" button click karo


STEP 3: Local Repository Ko GitHub Se Connect Karo
---------------------------------------------------
GitHub pe repository banne ke baad, yeh commands dikhenge. Copy karke terminal mein paste karo:

git remote add origin https://github.com/YOUR-USERNAME/minecraft-plugin-store.git
git branch -M main
git push -u origin main


ALTERNATIVE: Agar Git Install Nahi Hai
---------------------------------------
1. Git download karo: https://git-scm.com/download/win
2. Install karo (default settings se)
3. Computer restart karo
4. Phir upar ke steps follow karo
